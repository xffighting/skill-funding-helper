# Launch Checklist

- Confirm `.github/FUNDING.yml` points to public maintainer profiles.
- Confirm README support block does not duplicate old sponsor copy.
- Run `python3 scripts/skill_funding.py validate --repo-root .`.
- Open `index.html` and capture updated screenshots.
- Publish the GitHub repository.
- Enable Sponsorships in GitHub repository settings when available.
- Share `docs/wechat-article.md` with the generated images.
